1-Creacion De Base datos

Para crear base dato debe usar este comando:
create database nombre;

2-Mostrando Base datos Creada

para motras la base dato debe usar el siguente comando:
show databases;

3-creando la tabla de base datos usando primary key

para crear una tabla usando primary key debe usar este comado:
create table name base datos(
id int primary key,
nombre varchar(255),
edad int
);

4-Creando otra tabla usando primary key Auto Incremento

para crear una tabla con auto incremento se debe usar este comando: create table name table(
id int primary key auto_increment,
nombre varchar(225),
edad int
);

5-creando tabla con llave foraneas

ahora vamos hacer una conexion con llave foraneas aqui le dejo el comando : create table name table(
id int primary key,
nombre varchar(255)
);

entonces para hacer la conexion con la llave foraneas padre el comando que debe ejecutar es: CREATE TABLE name table (
id INT PRIMARY KEY,
nombre VARCHAR(255),
llaveforania INT,
FOREIGN KEY (llaveforania) REFERENCES llaveforania(id)
);
